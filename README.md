
# Razorpay Clone

Welcome to the Razorpay clone web application. This website is a static website developed by using TailwindCSS only.


## Tech Stack

HTML,
CSS,
TailwindCSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/harshika-bhise/Razorpay.git
```

Go to the project directory

```bash
  cd Razorpay
```

Install dependencies

```bash
  npm install
```
also need to install TailwindCSS or use CDN link of TailwindCSS



Start the server

```bash
  npm run start
```
